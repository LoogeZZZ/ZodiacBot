# 🔮 ZodiacBot

Telegram-бот, который генерирует **ежедневные матерные гороскопы** с помощью DeepSeek AI. Работает в личных чатах и закрытых группах.

## Возможности

- 🔮 Персональный гороскоп на каждый день для всех 12 знаков зодиака
- 🤖 Генерация текста через DeepSeek API
- 💾 Кэширование прогнозов в PostgreSQL — API не дёргается лишний раз
- ⏰ Автоматическое обновление гороскопов каждый день в 06:00 (cron)
- 👤 Профиль пользователя с сохранением знака зодиака
- 💬 Поддержка личных чатов и групповых чатов
- 🛠 Админ-панель для управления ботом

## Стек

- **Go** — основной язык
- **PostgreSQL** + `pgxpool` — база данных
- **DeepSeek API** — генерация текста
- **go-telegram-bot-api** — взаимодействие с Telegram
- **robfig/cron** — планировщик задач
- **godotenv** — управление переменными окружения

## Запуск

### 1. Клонируй репозиторий

```bash
git clone https://github.com/LoogeZZZ/ZodiacBot.git
cd ZodiacBot
```

### 2. Создай файл `.env`

```env
BOT_TOKEN=your_telegram_bot_token
DB_URL=postgres://user:password@localhost:5432/zodiacbot
DEEPSEEK_API_KEY=your_deepseek_api_key
```

### 3. Запусти базу данных

Убедись что PostgreSQL запущен и база создана.

### 4. Запусти бота

```bash
go run .
```

## Использование

**В личном чате:**
- `/start` — начать работу с ботом
- Выбери свой знак зодиака
- Нажми "🔮 Получить прогноз"

**В группе:**
- Упомяни бота `@your_bot_name` — получишь свой гороскоп
- `/change` — изменить знак зодиака
